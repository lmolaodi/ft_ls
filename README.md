# ft_ls

Description

shell ls command.

To recode ls command and some of its options.

Objectives

Recode the system's command ls.

Behavior must be identical to the original ls command.

ls command options
- ls
- ls -a
- ls -l
- ls -r
- ls -R
- ls -al

Compile Instructions

git clone repo link
cd ft_ls
run command make
